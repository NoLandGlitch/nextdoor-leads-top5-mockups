## Design System

Each business page has a distinct visual identity while sharing a common structural foundation:

- **Animal Clinic of Millville** — soft white / navy / muted teal; Stripe-inspired layout for clarity and trust
- **Ocean Heights Animal Hospital** — warm professional blue-green; serif headings (Merriweather); image-led sections
- **Atilis Gym EHT** — dark background / electric green / high contrast; condensed display headings (Barlow Condensed); intense energy
- **Jo Jo's Italian Grille** — warm cream / deep red / burnt orange; Playfair Display headings; photo-forward hospitality layout
- **Sharky's Sports Bar & Grill** — near-black / cyan / orange; dark Spotify-like UI; event and specials-first structure

Shared foundation:
- Semantic HTML5 throughout
- Responsive layout: mobile-first, desktop-optimized
- CSS custom properties for per-site theming
- Single shared stylesheet (`styles/main.css`)
- Single shared JS file (`scripts/main.js`) handling nav scroll, hamburger menu, today's-hours highlighting, and smooth scroll
- No build step — open `index.html` in a browser and everything works locally
